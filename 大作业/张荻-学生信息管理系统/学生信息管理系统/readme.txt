1.启动mongodb，运行create_col.py，创建数据库、集合。
2.运行start.py，进入学生管理系统界面
3.详细操作：
①检查连接（确认数据连接是否成功，确保后续操作顺利进行）
②全部学生（导出数据库里学生信息到ui界面）
③导出数据（导出数据库数据到test.txt）
④添加学生（右上角输入相关信息后点击添加学生）
⑤删除学生（清空全部按钮上方输入框输入学号后点击删除该学生）
⑥修改信息（右下角输入相应学生的学号，然后对姓名、年龄、班级、性别、籍贯这5项内容进行修改，最后点击修改信息）
⑦查找学生（右下角学号输入框输入学号后，点击查找学生）
⑧清空全部（只是清空ui界面学生信息）
